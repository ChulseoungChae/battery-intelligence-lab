# Battery Intelligence Lab

EV·ESS 배터리 데이터를 활용한 **데이터 분석 · 전처리 · 시각화 · AI 모델 학습** 연구용 저장소입니다.

충·방전 및 운용 로그를 바탕으로 배터리 상태(SOC/SOH)와 열화 경향을 파악하고, 머신러닝·딥러닝 모델로 예측·평가하는 실험을 모읍니다.

## 목표

- 배터리 데이터 전처리 및 품질 분석
- 충·방전 특성 · 열화 경향 시각화
- SOC / SOH / 잔여수명 예측 모델 개발
- 모델 학습·검증 및 성능 평가
- EV·ESS 등 데이터 소스 간 비교 분석
